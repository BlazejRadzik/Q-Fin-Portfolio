# Derivatives pricing app

> Streamlit: BS vs MC, zmiana $S$, $K$, $T$, $r$, $\sigma$ i liczby ścieżek.

## Uruchomienie

```bash
pip install streamlit numpy scipy pandas plotly
streamlit run app.py
```

## Dev Container

W VS Code: **Reopen in Container**, potem to samo polecenie `streamlit run app.py`.

## Testy

```bash
pytest
```
